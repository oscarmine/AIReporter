# AI Reporter

A desktop application for generating professional security vulnerability reports using AI.

## Overview

AI Reporter transforms raw vulnerability findings into polished, publication-ready security reports. It leverages Google's Gemini AI to generate comprehensive documentation including impact analysis, remediation steps, and severity assessments.

## Requirements

- Node.js v18 or higher
- Google Gemini API Key

## Installation

```bash
git clone https://github.com/oscarmine/AIReporter.git
cd AIReporter
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

The executable will be generated in the `release` folder *(Windows/mac/linux)*.

## Configuration

1. Open Settings
2. Enter your Gemini API Key
3. Adjust AI parameters as needed
